编程代码

对于代码，可以如下录入： 

<pre tag="langName">
    代码
</pre>

其中langName表示代码语言，下面一一列举支持的配置。

不设置

如果你不确定你的代码，可以不设置。 

<pre tag>
    // todo
</pre>

运行效果： 

    // todo

html 

<pre tag="html">
<h2>标题</h2>
</pre>

运行效果： 

标题

css 

<pre tag="css">
.cls{
    color: red;
    font-size: 12px;
}
</pre>

运行效果： 

.cls{
    color: red;
    font-size: 12px;
}

javascript 

<pre tag="javascript">
   console.log("你好呀！");
</pre>

运行效果： 

    console.log("你好呀！");

python 

<pre tag="python">
#!/usr/bin/python3
print("你好呀！")
</pre>

运行效果： 

#!/usr/bin/python3
print("你好呀！")

config 

<pre tag="config">
#定义服务器的一些行为
http {

    #默认文件类型
    default_type  application/octet-stream;
}
</pre>

运行效果： 

#定义服务器的一些行为
http {
    
    #默认文件类型
    default_type  application/octet-stream;
}
