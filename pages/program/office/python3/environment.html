环境和准备工作

安装Python3

我们来说明Windows电脑中如何安装。

打开 WEB 浏览器访问 https://www.python.org/downloads/windows/ ，页面截图如下：

上面截图显示，当前的最新文档版本（Stable Releases）是 Python 3.12.1 - Dec. 8, 2023

然后直接下载其中的 Download Windows installer (64-bit)，就可以了。下载好以后，和普通软件一样，进行安装即可。

在安装过程中，你会遇到类似下面的界面：

此时，一定要记得勾选类似 Add Python 3.6 to PATH 的选项哦，这非常重要。

安装依赖包

对于Windows电脑，直接打开CMD窗口：

然后在窗口输入下面命令并回车： 

    pip3 install -i https://mirrors.aliyun.com/pypi/simple/ py3office

运行效果大致如下：

如果后续的一些功能运行出错了，可能是py3office版本过低，可以运行下面命令进行升级即可： 

    pip3 install -i https://mirrors.aliyun.com/pypi/simple/ --upgrade py3office
