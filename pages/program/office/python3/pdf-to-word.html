PDF转word文件

单个文件

直接转Word

比如我们有一个pdf文件：demo.pdf，希望转成Word： 

#!/usr/bin/python3
# coding=utf-8

from py3office import pdf

pdf.toWord("./demo.pdf", "./")

运行之后，就可以看见当前目录下多了一份word文件：demo.docx。

修改Word名称

那如果想自定义生成后的word文件名称怎么办？比如word文件名称希望是文件.docx，修改代码为： 

#!/usr/bin/python3
# coding=utf-8

from py3office import pdf

def getName(name, texts, index, total):
    return "文件"
pdf.toWord("./demo.pdf", "./", getName)

from py3office import handler

    handler.printList(texts)

pdf.toWord("./demo.pdf", "./", getName)

比如我们想取[2] VISLite 贡献指南为word文件名称，那么代码就变成了： 

#!/usr/bin/python3
# coding=utf-8

from py3office import pdf
from py3office import handler

def getName(name, texts, index, total):
    return texts[2]
pdf.toWord("./demo.pdf", "./", getName)

这样，pdf就被转成了word文件：VISLite 贡献指南.docx。

批量处理

比如当前目录下有一个文件夹pdf，里面有很多pdf文件，目录结构如下： 

- pdf
    - 身份证（正面）.pdf
    - 身份证（正面）.pdf
    - 银行卡
        - 中国银行卡.pdf

我们希望把其中的所有pdf都变成word文件（保持原来的目录结构）： 

#!/usr/bin/python3
# coding=utf-8

from py3office import pdf
from py3office import handler

sourcePath = "./pdf"
targetPath = "./word"

def doit(fileInfo):
    if fileInfo["path"].endswith(".pdf"):

        def getName(name, texts, index, total):
            return name

        pdf.toWord(
            fileInfo["path"],
            fileInfo["folder"].replace(sourcePath, targetPath),
            getName,
        )


handler.listFile(sourcePath, doit)

运行后会在当前目录多一个文件夹word，目录结构如下： 

- word
    - 身份证（正面）.docx
    - 身份证（正面）.docx
    - 银行卡
        - 中国银行卡.docx
温馨提示：对于单个文件中对word文件名称的说明，这里类似的，直接修改getName返回值也是可行的。
为了更方便，你可以直接使用： pdf.logInfo("./demo.pdf", False) 来查看当前pdf文件内容。
