快速入门使用

基本说明

一般我们都需要安装至少两个（或更多）包来组合使用，这里我们举个例子： 

    npm install --save-dev @babel/core @babel/preset-env

其中 @babel/core 是核心包，必须的，而 @babel/preset-env 是具体干事的。

温馨提示：如果是ts，可以选择@babel/preset-typescript，如果是jsx，可以选择@babel/preset-react等，当然，他们之间也可以灵活的组合。

和webpack等配合使用的时候，可以通过配置文件来规范babel具体如何转义（下面还是以上面的包举例子）。

.babelrc 

{
    "presets": ["@babel/preset-env"]
}

babel.config.js 

module.exports = {
    "presets": ["@babel/preset-env"]
};

温馨提示：不同的插件有自己的可配置项，具体的查询对应的文档即可。

脚本代码

直接在需要的nodejs代码中使用下面的内容： 

const babelCore = require("@babel/core");

let code = babelCore.transformSync(source, {
    configFile: false,
    presets: [require("@babel/preset-env")]
}).code;

温馨提示：上面只是一个同步的例子，可以 点击此处 了解更多细节。
