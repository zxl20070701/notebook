Babel转义

安装并使用

首先，你需要安装必要的插件： 

npm install --save-dev @rollup/plugin-babel

然后，引入并使用： 

import babel from '@rollup/plugin-babel';

export default {
    plugins: [babel({
        babelHelpers:"bundled",

        babelrc:false,
        presets: ["@babel/preset-env"],

        include:["src/**"], // 只是例子，具体根据项目调整
        exclude:["node_modules/**"]

    })]
};

配置项

babelHelpers

有如下可选值：

  1. runtime ：如果你要用rollup构建一个js包的时候，使用该配置，该配置要结合@babel/plugin-transform-runtime插件使用，使用@babel/plugin-transform-runtime也要安装@babel/runtime插件；
  2. bundled ：如果用rollup构建一个项目的用此参数；
  3. inline ：官网不推荐使用，会导致很多重复性代码；
  4. external ：要结合@babel/plugin-external-helpers插件使用，它会把helpers 收集到一个共享模块，我的理解是相当于runtime和bundled一个集合，就是把helpers 抽到一个全局的共享模块但是又不会造成全局的污染。

babelrc

用于配置是否启用.babelrc文件来规范具体的babel配置。比如这里设置成false，就弃用了此文件，可以通过presets来指定。
