列表渲染

本质上来说，列表渲染其实就是生成一个数组，比如下面的代码： 

<ul>
    {
        [<li key="1">苹果</li>, <li key="2">香蕉</li>, <li key="3">橘子</li>, <li key="4">火龙果</li>]
    }
</ul>

渲染的时候就会变成： 

  • 苹果
  • 香蕉
  • 橘子
  • 火龙果

所以我们只需要返回一个数组即可实现，下面我们来举几个可行的例子。

Map 

    
    {["苹果", "香蕉", "橘子", "火龙果"].map((item, index) => <li key={ index }>{ item }</li>)}

For 

    
    {
        (() => {
            let result = [], list = ["苹果", "香蕉", "橘子", "火龙果"];

            for (var i = 0; i < list.length; i++) {
                result.push(<li key={ i }>{ list[i] }</li>)
            }

            return result
        })()
    }
