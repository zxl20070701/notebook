事件绑定

绑定事件

普通函数 

function doit() {
    alert('你点击了我')
}
class MyComponent extends React.Component {
    render() {
        return (<button onClick={doit}>点击我</button>)
    }
}

对象上的方法 

class MyComponent extends React.Component {
    doit() {
        alert('你点击了我')
    }
    render() {
        return (<button onClick={this.doit}>点击我</button>)
    }
}

传递参数

比如我们对象上有一个方法，点击触发的时候需要传递一个参数： 

class MyComponent extends React.Component {
    doit(info) {
        console.log('打印：'+info)
    }
    render() {
        return (<button onClick={() => this.doit.call(this, "来自按钮上的信息")}>点击我</button>)
    }
}
