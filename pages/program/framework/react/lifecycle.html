生命周期

挂载

当组件实例被创建并插入DOM中时会依次触发下面的历程：

constructor

在 React 组件挂载之前，会调用它的构造函数。

render

componentDidMount

在组件挂载后（插入DOM树中）立即调用。

更新

每当组件的state或props发生变化时，组件就会更新，而每一次改变都会重复经历如下历程：

shouldComponentUpdate

该方法会返回一个布尔值，指定React是否应该继续渲染，默认值是true，即每次state或props发生变化组件都会重新渲染。

render

componentDidUpdate

在更新后会被立即调用。

卸载

componentWillUnmount

在组件卸载及销毁之前直接调用。
