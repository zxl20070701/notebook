Props之数据传递

简单的理解就是：接收当前组件使用时候的属性值。看个例子： 

class Demo extends React.Component {
    constructor(props) {
        super(props)

        console.log(props)
    }
    render() {
        return 例子组件
    }
}

比如我们这样使用： 

render() {
    <Demo name="未命名" />
}

那么打印结果就是： 

    {name: '未命名'}

默认值

我们可以在定义完毕组件后，添加一些默认值： 

Demo.defaultProps = {
    key: 'demo-key'
}

打印结果就会变成： 

    {name: '未命名', key: 'demo-key'}
