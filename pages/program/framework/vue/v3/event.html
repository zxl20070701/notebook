事件event相关

以click为例，对于定义的组件，如果希望其上的click触发，比如： 

    <ui-demo @click=“doit” > </ui-demo>

如果组件内部什么也不干，是会触发原生事件的，而如果组件内部有类似如下语句： 

    let emit = defineEmits(["click"])

那除非组件内部再有类似如下语句： 

    emit(“click”)

来触发自定义click事件，否则什么也不会触发。

此外，组件内部的stop有效，比如组件里面的一个按钮大概如下： 

    <button @click.stop >按钮</button>

那点击这个按钮，什么也不会触发。
