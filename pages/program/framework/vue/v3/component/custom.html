如何定义组件？

defineComponent

文件index.vue： 

<template>
    <!-- 模板 -->  
</template>
<script lang="ts">  
import { defineComponent } from "vue";

export default defineComponent({
    setup() {
        // 代码
        return {};
    };
});
</script>  
<style lang="scss" scoped>  
    /* 样式 */
</style>

TSX

文件index.ts： 

import { defineComponent } from 'vue'
export default defineComponent({
    props: {},
    setup(props, ctx) {
        return () => {
            return (<div></div>)
        }
    }
});

ctx中包含：attrs、slots、emit、expose等内容。

此外，可以通过下面方法获取ctx： 

    let ctx = getCurrentInstance().ctx;

如此你就可以（不过里面不包含attrs、slots、emit、expose等，比如emit你可以直接引入defineEmits后创建获得）： 

    ctx.$forceUpdate();
