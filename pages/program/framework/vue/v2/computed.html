计算属性和侦听器

计算属性

基本写法 


<script>
    export default {
        computed: {
            info: function () {
                return "当前时间：" + new Date().valueOf()
            }
        }
    }
</script>

自定义setter

计算属性默认只有 getter，不过在需要时你也可以提供一个 setter： 

    export default {
        computed: {
            fullName:{
                get: function () {
                    return this.firstName + ' ' + this.lastName
                  },
                set: function (newValue) {
                    let names = newValue.split(' ')
                    this.firstName = names[0]
                    this.lastName = names[names.length - 1]
                }
            }
        }
    }

现在再运行 this.fullName = '托马斯 米奇' 时，setter 会被调用，this.firstName 和 this.lastName 也会相应地被更新。

侦听器

如果 `this.info` 发生改变，监听函数就会运行： 

    export default {
        data: {
            info:""
        },
        watch: {
            info: function (newQuestion, oldQuestion) {
                // todo
            }
        }
    }
