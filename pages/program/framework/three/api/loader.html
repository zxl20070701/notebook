模型加载器

GLTF加载器（GLTFLoader） 

var gltfLoader = new GLTFLoader(manager);
  • manager - 可选参数，默认为THREE.DefaultLoadingManager，该加载器将要使用的loadingManager

方法

load 

gltfLoader.load (url, onLoad, onProgress, onError);
  • url - 包含有.gltf/.glb文件路径/URL的字符串
  • onLoad - 加载成功完成后将会被调用的函数
  • onProgress - 可选参数，加载正在进行过程中会被调用的函数
  • onError - 可选参数，若在加载过程发生错误，将被调用的函数
