如何使用微信小程序原生组件？

准备好组件

比如我们有一个原生组件：wxexample，存放位置： 

- uniapp项目
    - wxcomponents
        - wxexample
            - index.js
            - index.json
            - index.wxml
            - index.wxss

比如上面的index.js，大概内容如下： 

Component({
    properties: {
        val: {
            type: "String",
            defalut: ""
        }
    },
    data: {},
    lifetimes: {
        ready() {}
    },
    methods: {}
});

引入

在pages.json文件中引入： 

{
    "globalStyle": {
		"mp-weixin": {
			"usingComponents": {
				"wxexample": "/wxcomponents/wxexample/index"
			}
		}
	}
}

使用

最后，在需要使用的地方使用即可： 

<!-- #ifdef MP-WEIXIN -->

<!-- #endif -->
