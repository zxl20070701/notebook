数据或事件绑定

数据绑定

我们在组件中定义一个数据： 

@Component({})
export class AppComponent {
    val = "默认值..."
}

使用的时候直接： 

    <input type="text" [value]="val">

运行截图：

当然，如果没有节点，也可以直接： 

    {{val}}

事件绑定

同样的，我们先定义一个方法： 

@Component({})
export class AppComponent {
    doit(event:Event){
        console.log(event)
    }
}

然后，在页面中就可以这样绑定了： 

    <input type="text" (input)="doit($event)">

上面绑定的是输入框的输入事件，如果我们修改输入框的内容就会看见控制台打印： 

    InputEvent {isTrusted: true, data: null, isComposing: false, …}
