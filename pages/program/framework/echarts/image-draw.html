Echarts环境准备

首先，需要把grid等边距设置好，比如这里我们归0（如果不设置0，别的地方对应调整即可）： 

grid: {
    left: 0,
    right: 0,
    top: 0,
    bottom: 0
}

然后，把水平垂直的原点设置为左下角，坐标计算的尺度和画布的宽和高保持一致： 

xAxis: {
    type: 'value',
    min: 0,
    max: el.clientWidth,
    show: false
},
yAxis: {
    type: 'value',
    min: 0,
    max: el.clientHeight,
    show: false
}

绘制

我们在使用ECharts绘制的时候，只需要配置坐标系统如下： 

coordinateSystem: "cartesian2d"

也就是计算坐标的时候使用xOy坐标系，而此刻xOy坐标系的尺寸又和图片保持了一致，因此上面采的点可以直接使用。

这非常类似 coordinateSystem: 'geo' 的情况。
