张量创建方法

基本用法

语法 

    tf.tensor(values, shape?, dtype?);

values表示数据，shape表示如何用数据values来组成张量，dtype表示数据类型。

温馨提示：dtype可以是'float32'|'int32'|'bool'|'complex64'|'string'，下同。

例子

比如我们创建一个矩阵： 

var myTensor = tf.tensor([1, 2, 3, 4, 5, 6], [3, 2], "int32");
myTensor.print();

打印结果： 

Tensor
  [[1, 2],
   [3, 4],
   [5, 6]]

当然，我们也可以这样写： 

var myTensor = tf.tensor([[1, 2], [3, 4], [5, 6]]);

可读&便捷写法

为了提高代码的可读性，我们有时推荐用更明确维度的方法来写。

tf.tensor[1|2|3|4|5|6]d

和tf.tensor语法都一样，只不过确定了维度，比如上面的例子这里可以改成： 

var myTensor = tf.tensor2d([[1, 2], [3, 4], [5, 6]]);

tf.scalar 

    tf.scalar(value, dtype?);

表示零维，很明显，只需要传递一个数据即可。

tf.ones 

    tf.ones(shape, dtype?);

值一定全是1，你只可以定义其结构。

tf.zeros 

    tf.zeros(shape, dtype?);

值一定全是0，你只可以定义其结构。
