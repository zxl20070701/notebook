变量的定义和改值

基本用法

语法 

    tf.variable(initialValue, trainable?, name?, dtype?);
  1. initialValue也就是用一个张量来初始化；
  2. trainable是布尔类型，默认true，可变；
  3. name是字符串类型，默认使用一个唯一的id；
  4. dtype数据类型。

例子 

var myTensor = tf.tensor2d([[1, 2], [3, 4], [5, 6]]);
    
var myVariable=tf.variable(myTensor);
myVariable.print();

打印结果： 

Tensor
  [[1, 2],
   [3, 4],
   [5, 6]]

tf.assign 

    tf.assign(newValue);

用于修改变量的值，比如我们对上面的例子补充下面的方法： 

myVariable.assign(tf.tensor2d([[7, 8], [9, 10], [11, 12]]));
myVariable.print();

打印结果： 

Tensor
  [[7, 8],
   [9, 10],
   [11, 12]]

不过，assign要求改变的值结构是一致的，比如我们下面的代码： 

    myVariable.assign(tf.tensor1d([1, 2, 3, 4]));

就会提示错误： 

Uncaught Error: shape of the new value (4) and previous value (3,2) must match
