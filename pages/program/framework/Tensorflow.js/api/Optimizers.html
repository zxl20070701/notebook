Optimizers 优化器

优化器算法列表

tf.train.sgd 

    tf.train.sgd(learningRate);

返回一个对象Optimizer。

参数

  • learningRate ：用于支持SGD算法的规定学习率，比如0.02，必选；

Optimizer

minimize 

    Optimizer.minimize(feedback, returnCost?, varList?);

参数

  • feedback ：一个函数，必输，指定要执行的函数及其输出要最小化；
  • returnCost ：布尔类型，指定是否返回执行 feedback() 产生的标量成本值；
  • varList ：它指定了可训练变量的列表。
