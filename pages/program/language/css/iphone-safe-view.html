iPhone中的安全显示区域

比如对底部小黑条做适配，我们可以这样： 

body{
    padding-bottom: constant(safe-area-inset-bottom); /* 兼容 iOS < 11.2 */
    padding-bottom: env(safe-area-inset-bottom); /* 兼容 iOS >= 11.2 */
}

一共有四个区域，分别是：

  1. safe-area-inset-left：安全区域距离左边边界的距离
  2. safe-area-inset-right：安全区域距离右边边界的距离
  3. safe-area-inset-top：安全区域距离顶部边界的距离
  4. safe-area-inset-bottom ：安全距离底部边界的距离
