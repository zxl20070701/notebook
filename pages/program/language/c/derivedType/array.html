数组

声明数组 

    type arrayName [ arraySize ];

这叫做一维数组。 arraySize必须是一个大于零的整数常量， type可以是任意有效的 C 数据类型。例如，要声明一个类型为 double 的包含 10 个元素的数组 balance，声明语句如下： 

    double balance[10];

初始化数组 

    double balance[5] = {1000.0, 2.0, 3.4, 7.0, 50.0};

大括号 { } 之间的值的数目不能大于我们在数组声明时在方括号 [ ] 中指定的元素数目。如果省略掉了数组的大小，数组的大小则为初始化时元素的个数，因此也可以： 

    double balance[] = {1000.0, 2.0, 3.4, 7.0, 50.0};

获取数组长度 

    int length = sizeof(balance);
