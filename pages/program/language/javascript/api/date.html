Date 日期

getFullYear()

语法 

    date.getFullYear()

可返回一个表示年份的4位数字。

getMonth()

语法 

    date.getMonth()

可返回表示月份的数字，返回值是 0（一月） 到 11（十二月） 之间的一个整数。

getDate()

语法 

    date.getDate()

可返回月份的某一天，返回值是 1 ~ 31 之间的一个整数。

getDay()

语法 

    date.getDay()

可返回一周（0~6）的某一天的数字，返回值是 0（周日） 到 6（周六） 之间的一个整数。

getHours()

语法 

    date.getHours()

可返回时间的小时字段，返回值是 0 （午夜） 到 23 （晚上 11 点）之间的一个整数。

getMinutes()

语法 

    date.getMinutes()

可返回时间的分钟字段，以本地时间显示，返回值是 0 ~ 59 之间的一个整数。

getSeconds()

语法 

    date.getSeconds()

可返回时间的秒，返回值是 0 ~ 59 之间的一个整数。
