基础语法

定义变量 

    var identifier type

或者一次声明多个变量： 

    var identifier1, identifier2 type

或者定义和赋值一气呵成： 

    identifier := value

同样可以多个变量一起： 

    identifier1, identifier2, identifier3 := value1, value2, value3

控制语法

if 语句 

if 布尔表达式 {
    /* 在布尔表达式为 true 时执行 */
}

例如： 

package main

import "fmt"

func main() {
    /* 定义局部变量 */
    var a int = 10
    
    /* 使用 if 语句判断布尔表达式 */
    if a < 20 {
        /* 如果条件为 true 则执行以下语句 */
        fmt.Printf("a 小于 20\n" )
    }
    fmt.Printf("a 的值为 : %d\n", a)
}

for 循环

Go 语言的 For 循环有 3 种形式。

和 C 语言的 for 一样 

    for init; condition; post { }

例如： 

package main

import "fmt"

func main() {
   sum := 0
      for i := 0; i <= 10; i++ {
         sum += i
      }
   fmt.Println(sum)
}

和 C 的 while 一样 

    for condition { }

例如： 

package main

import "fmt"

func main() {
    sum := 1
    for sum <= 10{
        sum += sum
    }
    fmt.Println(sum)
}

和 C 的 for(;;) 一样 

    for { }

例如： 

package main

import "fmt"

func main() {
   sum := 0
   for {
      sum++ // 无限循环下去
   }
   fmt.Println(sum) // 无法输出
}

循环嵌套

以下为 Go 语言嵌套循环的格式： 

for [condition |  ( init; condition; increment ) | Range]
{
   for [condition |  ( init; condition; increment ) | Range]
   {
      statement(s);
   }
   statement(s);
}

下面是九九乘法表的例子： 

package main 

import "fmt"

func main() {
    for m := 1; m < 10; m++ {
        /* fmt.Printf("第%d次：\n",m) */
        for n := 1; n <= m; n++ {
            fmt.Printf("%dx%d=%d ",n,m,m*n)
        }
        fmt.Println("")
    }
}

循环控制语句

序号 控制语句和描述
1
break 语句
经常用于中断当前 for 循环或跳出 switch 语句
2
continue 语句
跳过当前循环的剩余语句，然后继续进行下一轮循环。
3
goto 语句
将控制转移到被标记的语句。

goto 语句

语法格式如下： 

goto label;
..
.
label: statement;

下面的例子表示在变量 a 等于 15 的时候跳过本次循环并回到循环的开始语句 LOOP 处： 

package main

import "fmt"

func main() {
    /* 定义局部变量 */
    var a int = 10

    /* 循环 */
    LOOP: for a < 20 {
        if a == 15 {
            /* 跳过迭代 */
            a = a + 1
            goto LOOP
        }
        fmt.Printf("a的值为 : %d\n", a)
        a++    
    }  
}

函数

函数定义

Go 语言函数定义格式如下： 

func function_name( [parameter list] ) [return_types] {
    函数体
}

例如： 

/* 函数返回两个数的最大值 */
func max(num1, num2 int) int {
   /* 声明局部变量 */
   var result int

   if (num1 > num2) {
      result = num1
   } else {
      result = num2
   }
   return result
}

返回多个值 

package main

import "fmt"

func swap(x, y string) (string, string) {
   return y, x
}

func main() {
   a, b := swap("zxl20070701", "notebook")
   fmt.Println(a, b)
}

以上实例执行结果为： notebook zxl20070701 。
