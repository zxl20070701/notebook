安装nodejs

首先准备好nodejs源码包： 

    wget https://nodejs.org/dist/v14.20.0/node-v14.20.0.tar.gz

然后解压： 

tar -zxvf node-v14.20.0.tar.gz
cd node-v14.20.0

配置、编译、安装： 

./configure --prefix=/usr/local/node 
make && make install

温馨提示：如果make老失败，可以选择使用linux编译后的文件（node-v14.20.0-linux-x64.tar.gz），直接解压后使用。

配置nodejs环境： 

    vim /etc/profile

然后修改其中内容： 

#set nodejs env
export NODE_HOME=/usr/local/node
export PATH=$NODE_HOME/bin:$PATH
export NODE_PATH=$NODE_HOME/lib/node_modules:$PATH

重启生效： 

    source /etc/profile
