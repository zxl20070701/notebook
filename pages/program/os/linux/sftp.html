sftp

连接 

    sftp -P 22 username@ip

然后输入密码即可登录。

文件传输

上传文件 

    put -r /User/zxl20070701/Desktop/demo.html /root/zxl20070701

上面的例子就是把本机电脑的/User/zxl20070701/Desktop/demo.html文件上传到服务器的 /root/zxl20070701文件夹下。

下载文件 

    get -r /root/zxl20070701/demo.html /User/zxl20070701/Desktop

上面的例子就是把服务器的/root/zxl20070701/demo.html文件下载到本机电脑的 /User/zxl20070701/Desktop文件夹下。

服务器操作

  • pwd 打印服务器当前目录
  • cd 切换服务器当前目录
  • ls 列出服务器当前目录下的文件
  • mkdir 为服务器创建目录

本机操作

比服务器目录操作命令多了一个"l"(Local)：

  • lpwd 打印本机当前目录
  • lcd 切换本机当前目录
  • lls 列出本机当前目录下的文件
  • lmkdir 为本机创建目录
