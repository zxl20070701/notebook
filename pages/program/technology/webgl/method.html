常用方法&属性

基础

开启深度计算 

    gl.enable(gl.DEPTH_TEST);

缓冲区数据类型

可选的数据类型，应该是以下的某个： Uint8Array、Int16Array、Uint16Array、Int32Array、Uint32Array、Float32Array。

viewport

gl.viewport告诉WebGL如何将裁剪空间（-1 到 +1）中的点转换到像素空间 

    gl.viewport(0, 0, gl.canvas.width, gl.canvas.height);

当你第一次创建WebGL上下文的时候WebGL会设置视域大小和画布大小匹配，但是在那之后就需要你自己设置（当你改变画布大小就需要告诉WebGL新的视域设置）

普通画笔

点 

    gl.drawArrays(gl.POINTS, first, count);

直线 

    gl.drawArrays(gl.LINES, first, count);

连续直线 

    gl.drawArrays(gl.LINE_STRIP, first, count);

闭合直线 

    gl.drawArrays(gl.LINE_LOOP, first, count);

三角形 

    gl.drawArrays(gl.TRIANGLES, first, count);

共边三角形 

    gl.drawArrays(gl.TRIANGLE_STRIP, first, count);

旋转围绕三角形 

    gl.drawArrays(gl.TRIANGLE_FAN, first, count);

索引画笔

下面的参数type表示索引缓冲区的数据类型，应该是 gl.UNSIGNED_BYTE 或gl.UNSIGNED_SHORT。

gl.UNSIGNED_BYTE对应Uint8Array，gl.UNSIGNED_SHORT对应Uint16Array。

点 

    gl.drawElements(gl.POINTS, count, type, first);

直线 

    gl.drawElements(gl.LINES, count, type, first);

连续直线 

    gl.drawElements(gl.LINE_STRIP, count, type, first);

闭合直线 

    gl.drawElements(gl.LINE_LOOP, count, type, first);

三角形 

    gl.drawElements(gl.TRIANGLES, count, type, first);

共边三角形 

    gl.drawElements(gl.TRIANGLE_STRIP, count, type, first);

旋转围绕三角形 

    gl.drawElements(gl.TRIANGLE_FAN, count, type, first);
