npm script钩子
  • prepublish ：在npm publish命令之前运行。
  • prepublishOnly ：在npm publish命令前执行。
  • publish ：在npm publish命令后执行。
  • postpublish ：在npm publish命令后执行。
  • preinstall ：在npm install命令前执行。
  • postinstall ：在npm install命令后执行。
  • preuninstall ：在npm uninstall命令前执行。
  • postuninstall ：在npm uninstall命令后执行。
